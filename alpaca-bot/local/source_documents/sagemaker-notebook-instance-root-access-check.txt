<h1>sagemaker-notebook-instance-root-access-check<a name="sagemaker-notebook-instance-root-access-check"></a></h1>

<p>Checks if the Amazon SageMaker RootAccess setting is enabled for Amazon SageMaker notebook instances. The rule is NON_COMPLIANT if the RootAccess setting is set to ‘Enabled’ for an Amazon SageMaker notebook instance. </p>

<p><strong>Identifier:</strong> SAGEMAKER_NOTEBOOK_INSTANCE_ROOT_ACCESS_CHECK</p>

<p><strong>Resource Types:</strong> AWS::SageMaker::NotebookInstance</p>

<p><strong>Trigger type:</strong> Configuration changes</p>

<p><strong>AWS Region:</strong> All supported AWS regions except China (Beijing), Asia Pacific (Jakarta), Middle East (UAE), Asia Pacific (Hyderabad), Asia Pacific (Melbourne), AWS GovCloud (US-East), AWS GovCloud (US-West), Europe (Spain), China (Ningxia), Europe (Zurich) Region</p>

<p><strong>Parameters:</strong></p>

<p>None  </p>
