<h1>Sell Amazon SageMaker Algorithms and Model Packages<a name="sagemaker-marketplace-sell"></a></h1>

<p>Selling Amazon SageMaker algorithms and model packages is a three-step process:</p>

<ol>
<li><p>Develop your algorithm or model, and package it in a Docker container. For information, see <a href="sagemaker-marketplace-develop.md">Develop Algorithms and Models in Amazon SageMaker</a>.</p></li>
<li><p>Create an algorithm or model package resource in SageMaker. For information, see <a href="sagemaker-mkt-create.md">Create Algorithm and Model Package Resources</a>.</p></li>
<li><p>Register as a seller on AWS Marketplace and list your algorithm or model package on AWS Marketplace. For information about registering as a seller, see <a href="https://docs.aws.amazon.com/marketplace/latest/userguide/user-guide-for-sellers.html">Getting Started as a Seller</a> in the <em>User Guide for AWS Marketplace Providers</em>. For information about listing and monetizing your algorithms and model packages, see <a href="https://docs.aws.amazon.com/marketplace/latest/userguide/listing-algorithms-and-model-packages-in-aws-marketplace-for-machine-learning.html">Listing Algorithms and Model Packages in AWS Marketplace for Machine Learning</a> in the <em>User Guide for AWS Marketplace Providers</em>.</p></li>
</ol>
