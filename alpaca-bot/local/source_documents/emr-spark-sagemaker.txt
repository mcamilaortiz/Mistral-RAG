<h1>Use Amazon SageMaker Spark for machine learning<a name="emr-spark-sagemaker"></a></h1>

<p>When using Amazon EMR release version 5.11.0 and later, the <code>aws-sagemaker-spark-sdk</code> component is installed along with Spark. This component installs Amazon SageMaker Spark and associated dependencies for Spark integration with <a href="https://aws.amazon.com/sagemaker/">Amazon SageMaker</a>. You can use Amazon SageMaker Spark to construct Spark machine learning (ML) pipelines using Amazon SageMaker stages. For more information, see the <a href="https://github.com/aws/sagemaker-spark/blob/master/README.md">Amazon SageMaker Spark readme</a> on GitHub and <a href="https://docs.aws.amazon.com/sagemaker/latest/dg/apache-spark.html">Using Apache Spark with Amazon SageMaker</a> in the <em>Amazon SageMaker Developer Guide</em>.</p>
